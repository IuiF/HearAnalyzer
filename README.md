## 英語スクリプト表示アプリ
- 苦手問題表示






## pydubを使用しているためffmpegをシステムにインストールする必要があります

### windows
```
$ scoop install ffmpeg
$ choco install ffmpeg
```
### macOS
```
$ brew install ffmpeg
```
### Ubuntu/Debian
```
$ sudo apt-get update
$ sudo apt-get install ffmpeg
```
